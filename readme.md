*An efficient mandelbrot set render with 32b FP precision written in C++ with OpenGL shaders.*
Stunningly beautiful piece of mathematics.<br>

Controls:<br>
WASD -> Pan around.<br>
ARROWS -> Change C (unstable).<br>
R -> Reset C and order.<br>
I -> Increase order.<br>
O -> Decrease order.<br>
SHIFT -> Zoom out.<br>
SPACE -> Zoom in.<br>
